# SAR Highlights Extractor

自动提取出Super Animal Royale视频素材中的精彩片段（击倒或者击杀镜头）。

演示素材来自Genetorize。

使用了本代码制作的视频：<https://www.bilibili.com/video/BV1jg411V742>

## 运行演示

需要安装[FFmpeg](https://ffmpeg.org/download.html)和Python3。

```bash
python3 demo/main.py
```
